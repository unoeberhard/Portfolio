Portfolio

Inventory Management using HTML

Adamson University - May 24, 2024

×

Dadi's Store is an efficient inventory management system developed using HTML in Visual Studio, integrated with PHP, MySQL, and WampServer. It features a user-friendly interface that simplifies inventory operations, offering real-time updates and notifications on low stock levels directly on the dashboard. The system enables comprehensive monitoring of inventory products and maintains a detailed transaction history. With distinct user profiles for both cashiers and administrators, It ensures secure and role-based access to manage inventory and streamline operations effectively.

Arduino-Based Line Follower Robot

Adamson University - May 22, 2024

×

This project focuses on creating an Arduino-based Line Follower Robot using the LAFVIN robot kit. The robot is designed to autonomously navigate predefined paths by following black lines on a contrasting surface. Equipped with infrared sensors, the robot detects the path and adjusts its movement accordingly, demonstrating principles of robotics and automation. The project integrates Arduino programming and sensor interfacing to achieve precise control and navigation, making it an ideal platform for learning basic robotics concepts and sensor-based control systems.

YOLOVATOR: Elevator Head Count through YOLOv5

Adamson University - May 22, 2024

×

This project aims to optimize elevator efficiency and safety using YOLOv5-based camera systems for real-time head counting. By accurately monitoring passenger traffic and occupancy levels, the system minimizes wait times and congestion, ensuring a safer environment through continuous surveillance. Implemented with Arduino microcontrollers and AI algorithms, this initiative showcases improved elevator management, contributing to enhanced operational efficiency and safety protocols within educational settings.

TransitSense

Adamson University - May 18, 2024

×

TransitSense is an innovative software solution that optimizes public transportation by enhancing passenger flow through intelligent queue management. Using object detection technology, it identifies and analyzes real-time footage of commuters in crowded areas. This system improves resource allocation, streamlines boarding processes, reduces traffic congestion, and lowers carbon emissions, promoting efficient and sustainable urban transit.
